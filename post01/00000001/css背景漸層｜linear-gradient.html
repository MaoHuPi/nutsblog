css背景漸層｜linear-gradient

向下漸層

html:

<div class="linear1">向下漸層</div>

css:

.linear1 { background: linear-gradient(to bottom, yellow, orange); }

效果:

向下漸層



指定角度漸層

html:

<div class="linear2">指定角度漸層</div>

css:

.linear2 { background: linear-gradient(0deg, yellow, orange); }

效果:

指定角度漸層



對角線漸層

html:

<div class="linear3">對角線漸層</div>

css:

.linear3 { background: linear-gradient(to right bottom, yellow, orange); }

效果:

對角線漸層



指定範圍漸層

html:

<div class="linear4">指定範圍漸層</div>

指定範圍漸層

<div class="linear5">多層漸層</div>
